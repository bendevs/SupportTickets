# Support Tickets 

## How to use
-Clonar este repositiorio con __git clone__
- Copiar el archivo __.env.example__ a __.env__ e ingresar los datos para la conexion a la BD
- Ejecuta en una terminal en el folder del proyecto __composer install__
- Ejecuta en una terminal en el folder del proyecto __php artisan key:generate__
- Ejecuta en una terminal en el folder del proyecto __php artisan migrate__ 
- Ejecuta en una terminal en el folder del proyecto __npm install__
- Ejecuta en una terminal en el folder del proyecto __npm run build__ 
- Ejecuta en una termina en el folder del proyecto __php artisan serve__ 
- Puede registrarse como un usuario regular o loguearse como admin con  __admin@admin.com__ - __password__
